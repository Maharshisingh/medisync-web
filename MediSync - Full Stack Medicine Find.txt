MediSync - Full Stack Medicine Finder Application
📋 Project Overview
A comprehensive medicine finder platform connecting users with local pharmacies, featuring user authentication, medicine search, pharmacy partner portal, and admin dashboard.

🏗️ Project Architecture & Tech Stack
Phase 1: Backend Development
Core Technologies:
Runtime: Node.js (v18+)

Framework: Express.js

Language: JavaScript (converted from TypeScript)

Database: MongoDB Atlas (Cloud Database)

ODM: Mongoose (Object Document Mapping)

Authentication & Security:
JWT: JSON Web Tokens for authentication

bcryptjs: Password hashing and encryption

CORS: Cross-Origin Resource Sharing configuration

dotenv: Environment variable management

File Upload & Processing:
multer: File upload middleware for inventory management

XLSX processing: Excel file parsing for bulk inventory uploads
